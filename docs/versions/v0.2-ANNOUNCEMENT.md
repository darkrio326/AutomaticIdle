# Automatic Idle v0.2 发布公告

> 发布日期：2026-03-30

---

## 这一版带来了什么？

v0.2 在 v0.1 的自动化流水线基础上，补全了**进度门槛**和**动态目标**两块拼图——让玩家有事可做、有东西可买、有订单可追。

### 新增内容

**🏗️ 建筑系统**
购买建筑来解锁更多配方和资源链路。铁匠铺、采矿站……先解锁才能用。

**🔧 工具系统**（purchasable tools）
购买工具可加速对应配方的执行时间。合理投资，让流水线跑得更快。

**📦 动态订单**
每次最多 3 张随机订单，有时限、有冷却、有奖励。打破"只刷最优流程"的惯性，逼你做取舍。

**💾 流程模板**
保存最多 5 套流程配置，随时切换，不用每次手动重拼。

**🎮 欢迎页 + 路由**
独立欢迎页面，首次启动更有仪式感。路由切换干净利落。

### 体验优化

- 暗色主题全面修复：不再有白底块、亮色标签等降级显示
- 购买成功/失败提示改为全局居中浮层，不再挤在角落
- 快速配方进度条从"细线扫过"改为全宽流光动效，更直观
- 订单区域默认展开，不用手动点开

---

## 如何体验

```bash
git clone https://github.com/darkrio326/AutomaticIdle.git
cd AutomaticIdle
npm install
npm run dev
```

打开 http://localhost:5173，从欢迎页进入游戏。
