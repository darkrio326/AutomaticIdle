# AutomaticIdle v0.2

发布日期：2026-03-30

v0.2 在 v0.1 的自动化流水线基础上，补上了进度门槛和动态目标两块关键拼图。现在玩家不只是重复跑流程，而是需要决定先解锁什么、买什么工具、接不接订单、要不要切模板来追更高效率。

## 这一版带来了什么

### 建筑系统

购买建筑来解锁更多资源和配方链路。铁匠铺、采矿站等内容不再一开始全部开放，流程推进有了更明确的阶段感。

### 工具系统

工具变成可购买内容，并会直接影响对应配方的执行效率。你可以通过投资工具，让整条产线跑得更快。

### 动态订单

新增 3 槽位随机订单，带时限、冷却和奖励。它的作用不是单纯送钱，而是打断“只刷一套最优流程”的惯性，逼玩家做权衡。

### 流程模板

最多支持保存 5 套流程模板。常用产线可以随时切换，不需要每次都手动重拼。

### 欢迎页与路由

加入独立欢迎页，并完成欢迎页与游戏页的路由拆分，整体启动体验更完整。

## 体验优化

- 暗色主题显示问题做了系统性修复，不再出现白底块、亮色标签等降级样式
- 购买成功与失败提示改成更集中、更清晰的浮层反馈
- 快速配方进度条改成更直观的整条流动表现
- 订单区默认展开，减少额外点击
- 修复了工具效果不同步、引擎假运行、进度条异常等一批影响实际体验的问题

## 如何体验

```bash
git clone https://github.com/darkrio326/AutomaticIdle.git
cd AutomaticIdle
npm install
npm run dev
```

默认开发地址由 Vite 在终端输出。首次进入会先看到欢迎页，再进入主界面。
